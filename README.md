# mini-micro

My setup for the Mini Micro Fantasy Computer and my personal coding projects in MiniScript.

## repo layout

The following is the main repo files and directories:

- `data/` - My config directory including `bootOpts.grfon` and an extra `user.minidisk`
- `disk/` - My main `/usr` boot disk for programs and libraries
- `tmsim/` - A time machine simulator game
- `.claude/agents/micro-boi.md` - Claude agent with MiniScript and Mini Micro context

The following are Mini Micro configs and binaries are made possible by [Joe Strout](#credits):

- `MiniMicro.x86_64` - Mini Micro binary
- `UnityPlayer.so` - Unity game engine runtime
- `MiniMicro_Data` - Unity game configuration for Mini Micro

## ai disclaimer

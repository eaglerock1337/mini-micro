# mini-micro

My setup for the Mini Micro Fantasy Computer and my personal coding projects in MiniScript.

## repo layout

- `data/` - My config directory including `bootOpts.grfon` and an extra `user.minidisk`
- `disk/` - My main `/usr` boot disk for programs and libraries

## credits

Mini Micro binaries and data are provided by the [Mini Micro website](http://miniscript.org/MiniMicro) and the [MiniMicro2 project on GitHub](https://github.com/JoeStrout/MiniMicro2). Information about MiniScript is available at [miniscript.org](http://miniscript.org).
